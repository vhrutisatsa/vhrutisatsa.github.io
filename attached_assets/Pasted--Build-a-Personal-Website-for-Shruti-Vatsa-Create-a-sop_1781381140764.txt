# Build a Personal Website for Shruti Vatsa

Create a sophisticated personal website for an interdisciplinary researcher, writer, and artist.

The website should feel like a cross between:

* a research notebook
* a literary magazine
* an artist's journal
* an academic portfolio

It should NOT feel like:

* a startup landing page
* a SaaS product website
* a corporate portfolio
* a generic resume website

The overall impression should be intellectual, thoughtful, elegant, timeless, and personal.

---

## Identity

Name:
Shruti Vatsa

Tagline:

Researcher. Writer. Artist.

Alternative subtitle:

Exploring how humans search, learn, decide, and make meaning.

Short introduction:

I study how humans search for information, make decisions under uncertainty, and construct meaning. My work spans cognitive science, neuroscience, decision-making, information foraging, writing, and visual art. Through research, essays, and paintings, I explore the patterns that shape attention, agency, and experience.

---

## Design Philosophy

The design should be inspired by:

* research notebooks
* old books and journals
* literary magazines
* museum catalogues
* minimalist academic websites

Visual qualities:

* clean
* calm
* spacious
* typography-focused
* elegant
* highly readable

Avoid:

* excessive animations
* flashy effects
* glassmorphism
* neon colors
* startup aesthetics
* large marketing banners

The site should feel like a place where serious thinking happens.

---

## Color Palette

Primary colors:

* warm off-white background (#F8F5EF)
* charcoal text (#222222)
* muted forest green accents (#355E3B)

Optional secondary accents:

* deep burgundy
* muted navy

The color palette should be subtle and timeless.

---

## Typography

Use excellent typography.

Suggested pairings:

Headings:
Cormorant Garamond
or Playfair Display

Body:
Inter
or Source Sans Pro

Requirements:

* large readable body text
* generous line spacing
* comfortable reading width
* beautiful essay reading experience

The website should prioritize reading.

# TECHNICAL REQUIREMENTS

Use modern React and Next.js architecture.

Create reusable components.

Implement:

* markdown support for essays
* markdown support for notes
* image optimization
* SEO optimization
* RSS feed
* sitemap
* dark mode
* responsive design

---

# SPECIAL REQUIREMENT

The website must communicate one central idea:

Research, writing, and art are not separate activities.

They are different ways of exploring the same questions:

How do humans perceive, search, decide, learn, imagine, and create meaning?

Every design decision should reinforce that feeling.

The final website should feel closer to a personal intellectual home than a professional portfolio.
